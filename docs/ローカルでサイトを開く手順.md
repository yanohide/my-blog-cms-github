# ローカルでサイトを開く手順（初心者向け）

このサイトを自分のパソコンで表示するための手順です。プログラミングの知識がなくても大丈夫です。

---

## 準備するもの

- このプロジェクトのフォルダ（`my-blog`）がデスクトップにあること
- Node.js がインストールされていること（`npm run dev` が動くため）

---

## 手順

### ステップ1：ターミナルを開く

**ターミナル**は、パソコンに文字で命令を送るための画面です。

- **Mac の場合**：Spotlight（⌘ + スペース）で「ターミナル」と検索して開く
- **Cursor を使っている場合**：画面下の「ターミナル」タブをクリックするか、メニューから「ターミナル」→「新しいターミナル」を選ぶ

---

### ステップ2：プロジェクトのフォルダに移動する

ターミナルに次の文字を**そのままコピーして貼り付け**、Enter キーを押します。

```
cd /Users/sono/Desktop/my-blog
```

**何をしているか**：パソコンに「my-blog というフォルダで作業する」と伝えています。`cd` は「そのフォルダに移動する」という意味です。

---

### ステップ3：サイト用のサーバーを起動する

同じターミナルに、次の文字を**コピーして貼り付け**、Enter キーを押します。

```
npm run dev
```

**何をしているか**：サイトを表示するための小さなサーバー（プログラム）を起動しています。このサーバーが動いている間だけ、ブラウザでサイトを見られます。

**成功したときの表示例**：
```
✓ Ready in 2s
- Local: http://localhost:3000
```
このような表示が出たら、次のステップへ進んでください。

**このターミナルは閉じないでください。** 閉じるとサーバーが止まり、サイトが表示されなくなります。

---

### ステップ4：ブラウザでサイトを開く

1. **ブラウザ**（Chrome、Safari、Edge など）を開く
2. 画面上部の**アドレスバー**（URL を入力する欄）をクリックする
3. 次のアドレスを**コピーして貼り付け**、Enter キーを押す

```
http://localhost:3000
```

**何をしているか**：自分のパソコンの中で動いているサーバーに「サイトを見せて」とアクセスしています。`localhost` は「自分のパソコン」という意味です。

---

## まとめ（やること一覧）

| 順番 | やること | コマンド・操作 |
|------|----------|----------------|
| 1 | ターミナルを開く | アプリから「ターミナル」を起動 |
| 2 | プロジェクトフォルダに移動 | `cd /Users/sono/Desktop/my-blog` を入力して Enter |
| 3 | サーバーを起動 | `npm run dev` を入力して Enter |
| 4 | ブラウザでサイトを開く | アドレスバーに `http://localhost:3000` を入力して Enter |

---

## うまくいかないとき

### 「コマンドが見つかりません」と出る

- Node.js がインストールされていない可能性があります。  
  [Node.js の公式サイト](https://nodejs.org/)からインストールしてください。

### 「ポートが使用中です」と出る

- すでに別のターミナルでサーバーが動いている可能性があります。
- 他のターミナルを閉じるか、パソコンを再起動してから、もう一度ステップ2からやり直してください。

### サイトが真っ白、または「接続できません」と出る

- ステップ3のターミナルがまだ開いたままか確認してください。閉じているとサーバーが止まっています。
- `http://localhost:3000` の入力ミスがないか確認してください（`localhost` の `o` が数字の `0` になっていないかなど）。
